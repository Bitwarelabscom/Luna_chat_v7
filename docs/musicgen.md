# Suno Music Generation Guide

Suno AI uses "tags" in square brackets to guide music generation. These tags can influence song structure, style, mood, instrumentation, and vocal delivery.

## 1. Song Structure Tags
Place these tags on their own lines within your lyrics to define the song's progression.

- **[Intro]**: Musical introduction.
- **[Verse]**: Storytelling sections. Use **[Verse 1]**, **[Verse 2]**, etc.
- **[Pre-Chorus]**: Build-up before the chorus.
- **[Chorus]**: The main hook/refrain. Typically high energy.
- **[Post-Chorus]**: Short section following the chorus.
- **[Bridge]**: Contrasting section, often with new chords or mood.
- **[Breakdown]**: Minimalist instrumental section.
- **[Drop]**: EDM-style high-energy impact.
- **[Solo]**: Instrumental solo (e.g., **[Guitar Solo]**, **[Piano Solo]**, **[Saxophone Solo]**).
- **[Instrumental]**: Non-vocal section.
- **[Outro]**: Ending section.
- **[Coda]**: Extended ending or epilogue.
- **[End]**: Hard stop for the track.

## 2. Style & Genre Tags
Use these to set the overall sound of the track.

- **Broad Genres**: [Pop], [Rock], [Hip Hop], [EDM], [Jazz], [Country], [Metal], [R&B], [Classical], [Lo-fi].
- **Subgenres**: [Synthwave], [Phonk], [Metalcore], [Amapiano], [Trap], [Bluegrass], [Dream Pop], [Eurodance].
- **Decades**: [1950s], [60s], [70s], [80s], [90s], [2000s].
- **BPM/Tempo**: [Fast], [Slow], [128 BPM], [Double-time], [Half-time].

## 3. Mood & Energy Tags
Define the emotional state of the music.

- **Moods**: [Happy], [Melancholic], [Aggressive], [Dark], [Uplifting], [Epic], [Atmospheric], [Chill], [Spooky], [Romantic].
- **Energy**: [High Energy], [Low Energy], [Dynamic], [Quiet], [Loud].

## 4. Instrumentation Tags
Specify which instruments should be prominent.

- **Acoustic**: [Acoustic Guitar], [Piano], [Strings], [Violin], [Drums], [Cello], [Harp].
- **Electric**: [Electric Guitar], [Distorted Guitar], [Synth], [808 Bass], [Synthesizer], [Electric Piano].
- **Specialty**: [Banjo], [Fiddle], [Saxophone], [Trumpet], [Orchestra], [Choir].

## 5. Vocal & Delivery Tags
Guide how the AI should sing the lyrics.

- **Gender/Type**: [Male Vocal], [Female Vocal], [Deep Voice], [High Pitched].
- **Delivery Style**: [Rap Verse], [Spoken Word], [Whispered], [Belting], [Falsetto], [Anthemic], [Autotuned].
- **Harmony**: [Duet], [Backing Vocals], [Harmony], [Vocal Chop], [Ad-libs].

## 6. Advanced Tags & Tips
- **[Key Change]**: Modulate the key of the song.
- **[Build]**: Increase tension before a drop or chorus.
- **[Stop]**: Abrupt silence.
- **[Fade Out]**: Gradually decrease volume at the end.
- **Combining Tags**: You can combine tags like `[Chorus | 80s Synthpop | Female Vocal]`.

**Pro Tip**: For best results, start with an **[Intro]**, use clear **[Verse]** and **[Chorus]** blocks, and end with an **[Outro]** followed by **[End]**.
